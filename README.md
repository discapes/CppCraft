How to build on Fedora:
- Install the following packages:
	`glm-devel` 
	`mesa-libGL-devel`
	`glfw-devel` 
	`assimp-devel` 
	`stb_image-devel`
	`glad`
- Create glad files: `glad --profile core --generator c --api gl=4.6 --spec gl --out-path glad`
